## Capgemini Exceller 2026 (Complete Revamp)

### Pattern Overview
**Total Duration: 145+ minutes**
**Type: Non-Adaptive**
**No Negative Marking**
**Applicants: ~160,000 | Job Openings: ~120,000**

### Round 1: Online Assessment

#### Section 1: Technical MCQs + Pseudocode
- **40 questions, 40-50 minutes**
- **Difficulty: High**
- Topics:
  - Programming basics
  - Loops & conditionals
  - OOPs concepts
  - Output based questions
  - Pointers / references
  - DBMS basics
  - OS basics
  - Computer Networks basics
  - Simple DSA (arrays, strings)
- **NOTE: Traditional aptitude (quants/logical/verbal) REMOVED completely for Capgemini**

#### Section 2: English Communication
- **30 questions, 30 minutes**
- **Difficulty: High**
- Topics:
  - Sentence Correction
  - Prepositions
  - Grammar
  - Reading Comprehension
  - Synonyms & Antonyms
  - Speech and Tenses
  - Articles
  - Sentence Selection
  - Spotting Error
  - Sentence Completion

#### Section 3: Game-Based Cognitive Assessment
- **4 games out of 24 available**
- **20-30 minutes**
- **Difficulty: High**
- Games list:
  - Grid Challenge
  - Digit Challenge
  - Switch Challenge
  - Motion Challenge
  - Doesn't fit the rule
  - Inductive logical thinking
  - Deductive logical thinking
- Each game has 5-6 min time limit with demo before starting

#### Section 4: Behavioral / PowerSkills Assessment
- **100 questions**
- **20-25 minutes**
- Situation-based questions (agree/disagree)
- Same questions may be repeated in different forms to check consistency
- **WARNING:** Inconsistent answers = automatic fail

### Round 2: Coding Test
- **2 questions, 45 minutes**
- Languages: C, C++, Java, Python
- Packages based on performance:
  - Analyst (A4): 4.25 LPA (base, if fail coding)
  - Analyst Star (A4(P)): 5.75 LPA (clear 1 question)
  - Senior Analyst (A5): 7.50 LPA (clear both)

### Capgemini Summary for Mock Tests
**Traditional aptitude is GONE.** Our bank is largely irrelevant for Capgemini mocks except for English section.
**What we need for Capgemini mocks:**
- Pseudocode questions (40 needed)
- English questions (30 needed)
- Game simulations (4 types)
- Behavioral assessment (100 needed)
- Coding questions (2 needed)
